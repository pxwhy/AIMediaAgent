# 选题价值判断

## 使用时机

当 Agent 需要从采集内容中挑选值得进入创作流程的素材时使用。

## 输入要求

- raw_content_id
- 来源、分类、标题、链接
- 正文摘要或全文
- 采集时间、发布时间、互动或来源指标

## 执行步骤

1. 判断标题和正文是否包含明确事实、观点、变化或冲突点。
2. 判断内容是否具备可解释、可延展、可追踪的创作空间。
3. 判断内容是否仍有时效价值，过期信息降低分数。
4. 对信息不足、重复、纯公告、纯标题党内容谨慎选择。

## 输出要求

- selected 只在内容具备明确创作价值时为 true。
- score 按 0-100 给出，70 以上才建议进入后续创作。
- reason 说明选择或不选择的核心依据。
- data_limits 写明缺少正文、缺少来源或缺少指标等限制。

## 注意事项

不要因为标题看起来热闹就选择，必须能从内容中找到可写的信息基础。
