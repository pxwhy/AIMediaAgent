# 标题分析

## 使用时机

当用户需要分析自媒体作品标题结构、点击动机、关键词和标题优化方向时使用。
这个 Skill 适合和账号复盘、选题建议一起使用。

## 输入要求

需要尽量提供：
- 作品标题
- 对应阅读或播放量
- 点赞、评论、收藏等互动指标
- 作品主题或正文摘要

## 执行步骤

1. 将标题按数据表现分为高表现、一般表现和低表现。
2. 提取高表现标题中的关键词、冲突点、利益点、情绪点和悬念。
3. 分析低表现标题是否存在过泛、过长、信息弱、对象不清等问题。
4. 总结当前账号适合复用的标题结构。
5. 给出下一批内容可测试的标题方向。

关注维度：
- 标题是否清楚表达主题
- 是否有明确受众利益点
- 是否存在冲突、反差、悬念或情绪点
- 标题是否过泛、过长或信息密度不足
- 表现较好的标题有什么共同结构

## 输出格式

请按以下结构输出：

1. 高表现标题规律
2. 弱标题问题
3. 可复用标题公式
4. 标题改写示例
5. 下一批标题测试方向

## 注意事项

- 不要只说“标题吸引人”，要说明吸引点是什么。
- 标题公式必须可复用，不要只改写单个标题。
- 如果缺少数据，请基于标题文本做结构判断，并标注数据不足。
