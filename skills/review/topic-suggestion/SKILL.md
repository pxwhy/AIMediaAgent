# 选题建议

## 使用时机

当用户需要基于账号历史作品表现生成下一批选题、系列内容或标题方向时使用。

## 输入要求

需要尽量提供：
- 历史作品标题
- 正文摘要
- 互动指标
- 账号定位或目标方向，如果有

## 执行步骤

1. 找出表现较好的主题和内容切口。
2. 判断哪些主题可以系列化或延展。
3. 避免脱离账号已有内容突然转向。
4. 为每个选题说明适合原因和预期提升指标。
5. 给出标题方向和内容切入角度。

要求：
- 选题必须和账号已有内容有关联
- 优先延展表现较好的主题
- 避免给出过泛、无法执行的选题
- 每个选题都要说明为什么值得做
- 每个选题都要给一个标题方向

## 输出格式

每个选题按以下格式输出：

1. 选题名称
2. 推荐标题方向
3. 适合原因
4. 内容切入角度
5. 预期提升的数据指标

## 注意事项

- 不要给过泛选题，例如“分享经验”“做干货内容”。
- 标题方向不等于最终标题，可以给 1 到 2 个候选。
- 如果历史样本很少，请给实验型选题，并说明验证方式。
