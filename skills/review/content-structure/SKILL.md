# 内容结构分析

## 使用时机

当用户需要分析自媒体作品正文结构、开头吸引力、信息组织、完读体验和互动引导时使用。

## 输入要求

需要尽量提供：
- 作品标题
- 正文内容或较完整摘要
- 阅读、点赞、评论、收藏等指标
- 发布时间或状态，如果有

## 执行步骤

1. 检查开头是否快速给出问题、冲突、结论或场景。
2. 判断正文是否有清晰的段落递进。
3. 找出内容中的重复、跳跃、过度铺垫或结论不足。
4. 分析哪些结构可能提升完读、评论、收藏或转发。
5. 总结适合账号复用的正文模板。

关注维度：
- 开头是否快速进入主题
- 是否有明确的观点、故事或问题
- 内容段落是否有递进关系
- 信息是否重复、松散或跳跃
- 是否有足够的案例、细节和结论
- 结尾是否能形成评论、关注或转发动机

## 输出格式

请按以下结构输出：

1. 当前结构优点
2. 影响阅读体验的问题
3. 高表现内容的结构模式
4. 可复用正文模板
5. 下一步正文改写建议

## 注意事项

- 不要只评价文笔，要从结构和用户阅读路径分析。
- 建议要具体到开头、主体、结尾和互动引导。
- 如果正文缺失，请说明无法充分判断正文结构。
