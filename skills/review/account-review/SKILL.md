# 账号复盘

## 使用时机

当用户需要基于一个自媒体账号的历史作品、正文内容和互动数据做整体复盘时使用。
这个 Skill 负责总控复盘框架，不替代更细的标题、结构、受众、选题和动作类 Skill。

## 输入要求

至少需要：
- 账号平台、昵称或 UID
- 作品标题
- 作品正文或摘要
- 作品状态
- 阅读、点赞、评论、收藏、分享等可用指标

如果指标缺失，请明确说明数据不足，不要编造表现。

## 执行步骤

1. 先按整体账号定位判断内容方向是否清晰。
2. 对作品按主题、表达方式和数据表现进行分组。
3. 找出表现较好的作品共同点，并说明证据。
4. 找出表现较弱作品的问题模式，并说明可能原因。
5. 总结账号当前机会、风险和下一阶段重点。
6. 最后给出可执行动作，并说明观察指标。

重点关注：
- 账号当前内容定位是否清晰
- 作品之间是否存在稳定主题或风格
- 数据表现好的内容有哪些共同点
- 数据较弱的内容可能存在哪些问题
- 后续应该强化、减少或停止哪些内容方向

## 输出格式

请按以下结构输出：

1. 整体判断
2. 内容方向复盘
3. 高表现作品规律
4. 低表现作品问题
5. 下一阶段策略
6. 可执行动作

## 注意事项

- 不要只复述数据，要解释原因。
- 不要给泛泛建议，例如“提升内容质量”。
- 每个重要结论都要尽量引用作品标题或指标作为依据。
- 如果样本量太少，结论要标注为初步判断。
