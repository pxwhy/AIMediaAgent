# 受众洞察

## 使用时机

当用户需要从作品主题、表达方式和互动数据中推断账号受众兴趣、需求和内容匹配度时使用。

## 输入要求

需要尽量提供：
- 作品标题
- 正文或摘要
- 阅读、点赞、评论、收藏、分享等指标
- 评论内容，如果有

## 执行步骤

1. 从标题和正文推断作品面向的核心人群。
2. 对比不同主题和表达方式的数据表现。
3. 判断受众更偏好知识、经验、情绪、观点还是实用方案。
4. 提炼受众尚未被满足的需求。
5. 给出内容定位和受众匹配度判断。

关注维度：
- 作品主要吸引哪类人群
- 受众更关心知识、情绪、经验、观点还是实用方案
- 哪些内容更容易引发点赞、评论或收藏
- 受众可能有哪些未被满足的需求
- 账号当前内容和目标受众是否匹配

## 输出格式

请按以下结构输出：

1. 核心受众画像
2. 受众兴趣点
3. 可深挖需求
4. 当前内容与受众匹配度
5. 不建议继续投入的方向

## 注意事项

- 受众画像要基于作品和数据推断，不要凭空设定。
- 结论要区分“确定性较高”和“需要继续验证”。
- 如果缺少评论数据，不要假装知道评论区反馈。
