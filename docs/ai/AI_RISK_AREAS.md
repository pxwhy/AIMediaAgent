# AI Risk Areas

## 高风险模块

### accounts

风险：

- Cookie 泄露
- 登录态失效
- 账号被平台风控

要求：

- 敏感数据不能进前端日志。
- Cookie 只能由账号模块读写。

### publisher

风险：

- 自动发布失败
- 平台页面变化
- 验证码和风控

要求：

- 必须有超时。
- 必须有失败状态。
- 不允许绕验证码。

### agent

风险：

- 生成虚假内容
- 内容违规
- API Key 泄露

要求：

- 发布前必须进入审核流。
- 模型调用失败不能伪装成功。

### image service

风险：

- 图片下载卡死
- 坏图导致任务崩溃
- 版权风险

要求：

- 下载必须有 timeout。
- 图片必须校验格式。
- 图片不足时进入人工补图或失败状态。

### config

风险：

- API Key 明文散落
- 本地和生产配置混淆

要求：

- 配置集中在 `core/config.py`。
- `.env` 不提交。
