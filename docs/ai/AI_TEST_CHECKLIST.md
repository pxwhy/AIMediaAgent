# AI 测试清单

## 启动与基础联通

- `GET /health` 返回 `{"status":"ok"}`。
- 前端 `npm run build` 通过。
- 启动脚本能拉起 API 和 H5。
- API 不可用时前端能显示明确失败状态。

## 账号与登录

- 可以创建头条号账号。
- 可以创建小红书登录会话。
- 登录确认成功后会创建或更新账号。
- 同平台同 UID 重复登录不会生成重复账号。
- 单个删除账号后列表正确刷新。
- 批量删除账号后列表、选中态、关联表单值正确清空。

## 账号作品同步

- 头条号账号可以触发同步作品。
- 同步后能看到作品标题、时间、状态。
- 作品详情弹框能看到正文内容。
- 同步不会把栏目页、收益页等无效项混入作品列表。
- 再次同步不会重复插入相同作品。

## 采集与素材库

- 可以拉取来源列表和分类。
- 可以预览热点内容。
- 可以把热点导入素材库。
- 素材按 `source_url` 去重。
- 素材详情弹框中图片优先展示，正文不会遮挡图片。
- 素材删除后列表和选择态正确更新。

## 智能筛选

- 可以选择账号、复盘、肖像、挑选 Agent。
- 能触发智能筛选并返回结构化结果。
- 推荐/不推荐切换筛选正常。
- 智能筛选结果可以导入素材库或复用已有素材。
- 没有足够依据时不应伪造高置信度选择。

## 模型配置

- 可以新增 DeepSeek 模型。
- 可以新增 OpenAI-compatible 模型。
- 可以测试模型并看到响应内容。
- 可以设置全局默认模型。
- API Key 未配置时列表显示明确状态。
- Base URL 填错时能看到原始可读错误。

## Agent 与 Skills

- 可以新增 Agent。
- 可以给 Agent 绑定模型和本地 Skill。
- 可以设置默认 Agent。
- 可以扫描本地 Skills 目录并加载 `SKILL.md`。
- Skills 查看详情使用弹框。
- Agent 保存后复盘/肖像/智能筛选页能正确读取配置。

## 账号复盘

- 选择账号后可以生成复盘。
- 可以查看复盘历史。
- 历史查看使用弹框。
- 删除复盘历史后状态和列表同步刷新。
- 复盘输出应包含 summary、定位、优质作品、选题建议、行动建议等关键字段。

## 账号肖像

- 选择账号后可以生成肖像。
- 可选绑定复盘结果作为输入。
- 可以查看肖像历史。
- 历史查看使用弹框。
- 删除肖像历史后状态和列表同步刷新。
- 肖像输出应包含定位、受众、内容赛道、来源偏好、禁区和发布建议。

## 发布任务

- 可以从素材创建草稿和发布任务。
- 可以打开头条号编辑器。
- 可以打开小红书编辑器。
- 可以执行自动发布。
- 发布失败时能看到详情弹框里的错误日志、截图、平台返回。
- `mark-published`、`mark-failed`、`cancel` 状态更新正确。

## 头条号自动发布

- 自动发布前能正确进入编辑页。
- 自动发布时会勾选“投放广告赚收益”。
- 发布完成后任务状态和发布时间正确回写。

## 小红书自动发布

- 能通过宿主节点进入 Shadow DOM 查找发布按钮。
- 不能依赖固定分辨率点击作为正式路径。
- 发布按钮、正文输入、图片处理在不同窗口尺寸下仍可工作。

## 安全与边界

- `.env` 不提交。
- API Key 不出现在前端日志。
- Cookie / storage_state 不返回给非必要接口。
- Agent 不读取 Cookie。
- Publisher 不调用模型。
- H5 不直接读写数据库或本地状态文件。
