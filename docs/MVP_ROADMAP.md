# 路线图

## 已完成

1. FastAPI 后端骨架与核心数据表。
2. Vue3 工作台主界面。
3. 账号登录会话接入，支持头条号、小红书。
4. 账号按 `platform + uid` 去重。
5. 账号单删、前端批量删除。
6. 头条号作品同步，支持标题和正文回收。
7. 热点采集预览、导入素材库、素材去重。
8. 模型配置管理、模型测试、默认模型切换。
9. Agent 管理与本地 Skills 绑定。
10. 账号复盘、账号肖像、智能筛选三类 AI 能力。
11. 头条号、小红书发布任务流与自动发布诊断。
12. 工作台列表分页、详情弹框化改造。

6. 按模块拆分后端统一路由文件。
